## ðŸ’» Executing

### :computer_mouse: Backend: 

```bash
# Clone this project
$ git clone https://github.com/italo-lima/omnistack7.git
# Join folder
$ cd backend
# Install dependencies
$ yarn 
# Run the project
$ yarn start 
```

### :computer: Frontend: 

```bash
# Join folder
$ cd frontend 
# Install dependencies
$ yarn 
# Run the project
$ yarn start
```

### :iphone: Mobile:

```bash
# Join folder
$ cd mobile 
# Install dependencies
$ yarn 
# Android 
$ react-native run-android
# iOS 
$ react-native run-ios
```
